\section{Trick 1: $\Big ( \frac{1}{n} \Big ) \Big ( \frac{1}{n+1} \Big ) = \frac{1}{n} - \frac{1}{n+1}$}
\label{a_equals_b_equals_1}

This incredibly useful bit of mathematical magic works when the denominators of the two fractions are of the form $n$ and $n+1$, their numerators are both 1, and $n > 0$. 
For example, something like 

\bigskip
\begin{equation*}
\left (\frac{1}{3} \right ) \left ( \frac{1}{4} \right ) = \frac{1}{12} \text{ and } \frac{1}{3} -  \frac{1}{4} = \frac{4 -3}{12} =  \frac{1}{12} 
\end{equation*}

\bigskip
\noindent
So why does this work? Consider


\begin{equation*}
\begin{array}{lllll}
{\displaystyle \bigg ( \frac{1}{n} \bigg ) \bigg ( \frac{1}{n+1} \bigg )}              
&=& {\displaystyle \frac{1}{n \cdot (n+1)}}                                                  &\qquad \qquad \mathrel{\#} \text{numerators are both 1} \\
[10pt]
&=& {\displaystyle \frac{(n-n) + 1}{n \cdot (n+1)}}                                      &\qquad \qquad \mathrel{\#} \text{$(n-n) = 0$ and  $0+1 = 1$} \\
[10pt]
&=& {\displaystyle \frac{(n+1) - n}{n \cdot (n+1)}}                                      &\qquad \qquad \mathrel{\#} \text{rearrange (+ is commutative and associative)} \\
[10pt]
&=& {\displaystyle \frac{(n+1)}{n \cdot (n+1)} - \frac{n}{n \cdot (n+1)}}      &\qquad \qquad \mathrel{\#} \text{split into a difference} \\
[10pt]
&=& {\displaystyle \frac{1}{n} - \frac{n}{n \cdot (n+1)}}                              &\qquad \qquad \mathrel{\#} \text{cancel $(n+1)$ in the first term} \\
[10pt]
&=& {\displaystyle \frac{1}{n} - \frac{1}{n+1}}                                            &\qquad \qquad \mathrel{\#} \text{cancel $n$ in the second term $\square$} 
\end{array}
\end{equation*}


\bigskip
\noindent
So we have 


\begin{equation}
\bigg ( \frac{1}{n} \bigg ) \bigg ( \frac{1}{n+1} \bigg ) =  \frac{1}{n} - \frac{1}{n+1}
\label{eqn:trick_1}
\end{equation}

\medskip
\bigskip
\noindent
An example of the use of this trick in evaluating infinite series is Burkard Polster's (aka Mathologer) use of Equation \ref{eqn:trick_1} when describing how 
Euler's solution to the Basel problem works \cite{mathologer:eulers_real_identity}. 

\bigskip
\subsection{Telescoping Series}
\label{subsec:telescoping_series}
What is a telescoping series? Briefly, a telescoping series is a sum that is characterized by the fact that the partial sums of the series (called telescoping sums) 
contain pairs of consecutive terms which cancel each other, leaving only the first and final terms. This cancellation of adjacent terms is whimsically 
called "collapsing the telescope".

\bigskip
\noindent
For example, consider the infinite series whose terms are described by Equation \ref{eqn:trick_1}. This is a telescoping series, which can
be seen in the following

%
% the comments are really hacked up in the following since one of the lines is too long to allow the array comments
%
\begin{equation*}
\begin{array}{lllll}
{\displaystyle \sum\limits_{n=1}^\infty \left ( \frac{1}{n} \right ) \left ( \frac{1}{n+1} \right )}
&=& {\displaystyle \sum\limits_{n=1}^\infty \left( \frac{1}{n} - \frac{1}{n+1} \right)}   \qquad \qquad \qquad \qquad \mathrel{\#} \text{Equation \ref{eqn:trick_1} (this is trick 1)} \\
[12pt]
&=& {\displaystyle \lim_{N \to \infty} \sum_{n=1}^N \left( \frac{1}{n} - \frac{1}{n+1} \right)} \:\; \quad \qquad \qquad \mathrel{\#} \text{we are interested in the limit} \\
[12pt]
&=& {\displaystyle \lim_{N \to \infty} \left\lbrack {\left(1 - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \cdots + \left(\frac{1}{N} - \frac{1}{N+1}\right) } \right\rbrack}  \\
[12pt]
&=& {\displaystyle \lim_{N \to \infty} \left\lbrack {  1 + \left( - \frac{1}{2} + \frac{1}{2}\right) + \left( - \frac{1}{3} + \frac{1}{3}\right) + \cdots + \left( - \frac{1}{N} + \frac{1}{N}\right) - \frac{1}{N+1} } \right\rbrack} \\
[12pt]
&=& {\displaystyle \lim_{N\ \to \infty} \left\lbrack {  1  - \frac{1}{N+1} } \right\rbrack}      \; \quad \qquad \qquad \qquad \mathrel{\#} \text{collapse the telescope} \\
[10pt]
&=& 1            
\end{array}
\end{equation*}

\bigskip
\noindent
A bit more formally, a telescoping series is the sum of terms of the form $(a_j - a_{j+1})$. As a result of the 
structure of its terms a telescoping series has the following property:

\begin{equation*}
\begin{array}{lllll}
{\displaystyle \sum\limits_{j = 1}^{n} (a_{j} - a_{j+1})}
&=&  (a_1 - a_2) + (a_2 - a_3) + (a_3 - a_4) + \cdots + (a_n - a_{n+1})             &\quad \mathrel{\#} \text{expand terms} \\
[12pt]
&=& a_1 + (a_2 - a_2) + (a_3 - a_3) + \cdots + (a_n - a_n) - a_{n+1}                 &\quad \mathrel{\#} \text{group like terms} \\
[12pt]
&=& a_1 - a_{n+1}                                                                                                &\quad \mathrel{\#} \text{simplify} \\
\end{array}
\end{equation*}

\bigskip
\subsection{Triangular Numbers}
The example I want to explore here is the infinite sum of the reciprocals of the triangular numbers \cite{wiki:triangle_numbers}.  So first,
what is a triangular number? The first six triangular numbers are depicted as triangles in Figure \ref{fig:triangle_numbers}. We can see 
from the figure that a triangular number $T_n$ is the sum of the integers from 1 to $n$. 
  
\bigskip
\bigskip
\begin{figure}[H]
\center{\includegraphics[frame,scale=0.35] {images/first_six_triangular_numbers.png}}
\caption{The First Six Triangular Numbers \cite{wiki:triangle_numbers}}
\label{fig:triangle_numbers}
\end{figure}

\bigskip
\noindent
Since $T_n$ is just the sum of the integers from 1 to $n$ we can easily calculate the first few values of $T_n$, which are
1, 3, 6, 10, 15, 21, 28, 36, 45, 55, 66, 78, 91, 105 and so on (note that some authors include 0 in the triangular number sequence).

\bigskip
\noindent
Ok, but what is the general form of $T_n$?  Since $T_n$ is the sum of the first $n$ integers 
we know that

\begin{equation}
T_{n} = \sum _{k=1}^{n} k=1+2+3+\dotsb +n = {\frac {n(n+1)}{2}} = {n+1 \choose 2}
\label{eqn:T_n}
\end{equation}

\bigskip
\noindent
Interestingly if you start with the number 1 in row 2 of Pascal's Triangle \cite{wiki:pascals_triangle} the corresponding diagonal contains the 
triangular numbers. This is shown in blue in Figure \ref{fig:pascals_triangle}.

\bigskip
\bigskip
\begin{figure}[H]
\center{\includegraphics[frame,scale=0.34] {images/pascals_triangle.jpg}}
\caption{Number Sequences Found In Pascal's Triangle}
\label{fig:pascals_triangle}
\end{figure}


\bigskip
\noindent
As an aside, the German mathematician and scientist Carl Friedrich Gauss is said to have discovered the consecutive integer formula,
${\displaystyle \sum _{k=1}^{n} k = \frac {n(n+1)}{2}}$, as a young person; however, this history is at the very least 
controversial \cite{wiki:triangle_numbers}.

\subsection{So What is the Sum of the Reciprocals of the Triangular Numbers?}
\bigskip
\noindent
All of this is good, but back to the question: what is the sum of the reciprocals of the triangular numbers? Well, first, the sum looks like

\bigskip
\begin{equation*}
\begin{array}{lllll}
{\displaystyle \sum \limits_{n=1}^{N} \frac{1}{T_n}}
&=& {\displaystyle  \sum _{n=1}^{N} \frac{1}{\frac{n(n+1)}{2}}}                                                &\qquad \qquad \mathrel{\#} \text{definition of $T_n$ (Equation \ref{eqn:T_n})} \\
[12pt]
&=& {\displaystyle \sum _{n=1}^{N} \frac{2}{n (n+1)}}                                                              &\qquad \qquad \mathrel{\#} \text{simplify} \\
[12pt]
&=& {\displaystyle  2 \sum _{n=1}^{N} \left (\frac{1}{n} \right) \left ( \frac{1}{n+1} \right )}        &\qquad \qquad \mathrel{\#} \text{pull 2 out of the sum and group terms} \\
[12pt]
&=& {\displaystyle  2 \sum\limits_{n=1}^N \left( \frac{1}{n} - \frac{1}{n+1} \right)}                    &\qquad \qquad \mathrel{\#} \text{Equation \ref{eqn:trick_1}}
\end{array}
\end{equation*}

\bigskip
\noindent
We saw in Section \ref{subsec:telescoping_series} that 
${\displaystyle {\lim_{N \to \infty} \sum_{n=1}^N \left( \frac{1}{n} - \frac{1}{n+1} \right)} = 1}$ and we know from the above that the infinite sum of the reciprocals of the triangular 
numbers is twice this  limit. So we know that 

\bigskip
\begin{equation*}
\sum \limits_{n=1}^{\infty} \frac{1}{T_n} = {\displaystyle 2 \left [\lim_{N \to \infty} \sum_{n=1}^N \left( \frac{1}{n} - \frac{1}{n+1} \right) \right ] = 2 \cdot 1 =  2}
\end{equation*}


\bigskip
\subsection{A Slightly More General Form Of This Trick}
\label{sec:trick_1_generalization}
A  slightly more general form of this trick allows for denominators of the form $n$ and $n+k$ rather than the $n$ and $n+1$ denominators we 
saw in Section \ref{a_equals_b_equals_1}. The constraints on this form of the trick are that the denominators are of the form $n$ and $n+k$, 
the product of the numerators must equal the absolute value of the difference of the denominators (that is, $k$), and $n > 0$.

\bigskip
\noindent
If we consider the case in which $k = ab$ with $a \geq 0$, $b \geq 0$, and $n > 0$ we see that

\bigskip
\begin{equation*}
\begin{array}{lllll}
{\displaystyle \bigg ( \frac{a}{n} \bigg ) \bigg ( \frac{b}{n+ab} \bigg )}              
&=& {\displaystyle \frac{ab}{n \cdot (n+ab)}}                                                  &\qquad \qquad \mathrel{\#} \text{the product of the numerators $ab = | n - (n+ab)|$} \\
[12pt]
&=& {\displaystyle \frac{(n-n) + ab}{n \cdot (n+ ab)}}                                     &\qquad \qquad \mathrel{\#} \text{$(n-n) = 0$ and $0+ab = ab$} \\
[12pt]
&=& {\displaystyle \frac{(n+ab) - n}{n \cdot (n+ab)}}                                      &\qquad \qquad \mathrel{\#} \text{rearrange (+ is commutative and associative)} \\
[12pt]
&=& {\displaystyle \frac{(n+ab)}{n \cdot (n+ab)} - \frac{n}{n \cdot (n+ab)}}    &\qquad \qquad \mathrel{\#} \text{split into a difference} \\
[12pt]
&=& {\displaystyle \frac{1}{n} - \frac{n}{n \cdot (n+ab)}}                                 &\qquad \qquad \mathrel{\#} \text{cancel $(n+ab)$ in the first term} \\
[12pt]
&=& {\displaystyle \frac{1}{n} - \frac{1}{n+ab}}                                               &\qquad \qquad \mathrel{\#} \text{cancel $n$ in the second term $\square$}
\end{array}
\end{equation*}

\bigskip
\bigskip
\noindent
So for $a \geq 0$, $b \geq 0$, and $n > 0$ we have 

\bigskip
 \begin{equation}
 {\displaystyle \bigg ( \frac{a}{n} \bigg ) \bigg ( \frac{b}{n+ab} \bigg ) =  \frac{1}{n} - \frac{1}{n+ab}}
 \label{eqn:trick_1_generalization}
 \end{equation}

\bigskip
\bigskip
\noindent
To see how this works consider the following example: let  $a = 2$,  $b = 5$  and $n = 3$. Then

\bigskip
\bigskip
\begin{equation*}
{\displaystyle \left ( \frac{2}{3} \right ) \left ( \frac{5}{3 +(2 \cdot 5)} \right ) = \left ( \frac{2}{3} \right ) \left ( \frac{5}{3 +10} \right ) = \left ( \frac{2}{3} \right ) \left ( \frac{5}{13} \right ) = \frac{10}{39}}
\end{equation*} 

\bigskip
\noindent
and

\begin{equation*}
{\displaystyle \frac{1}{3} - \frac{1}{3+ (2 \cdot 5)}  = \frac{1}{3} - \frac{1}{3+ 10} = \frac{1}{3} - \frac{1}{13} = \frac{13}{39} - \frac{3}{39} = \frac{13-3}{39}  = \frac{10}{39}}
\end{equation*}

\bigskip
\bigskip
\bigskip
\noindent
We can also note that if $a = b = 1$ in Equation \ref{eqn:trick_1_generalization} we get the special case we considered in Section \ref{a_equals_b_equals_1}:

\bigskip
\begin{equation*}
 {\displaystyle \left ( \frac{1}{n} \right ) \left ( \frac{1}{n+(1 \cdot 1)} \right ) = \left ( \frac{1}{n} \right ) \left ( \frac{1}{n+1} \right ) =  \frac{1}{n} - \frac{1}{n+(1 \cdot 1)}  = \frac{1}{n} - \frac{1}{n+1}}
 \end{equation*}
 
 \bigskip
 \bigskip
 \noindent
 Equation \ref{eqn:trick_1_generalization} also works correctly for $a = 0$ and $b \neq 0$, $a \neq 0$ and $b = 0$, and  $a = b = 0$.

\bigskip
\noindent
Is this generalization useful? I don't know but it was fun trying to make it work.

 
 \bigskip
 \section{Conclusions}
I'm not sure if Equation \ref{eqn:trick_1_generalization} is useful for anything but it was a fun exercise to do. On the other hand, 
the basic trick (Equation \ref{eqn:trick_1}) turns out to be very useful, especially when evaluating infinite telescoping series.
