\section{Introduction}
One way to think about this question is the following: 

\begin{equation*}
\begin{array}{llll}
S
&=&  \sqrt{2 + \sqrt{2 + \sqrt{2 + \sqrt{2 + \cdots}}}} 
				&\qquad \qquad \mathrel{\#} \text{define $S$} \\
[5pt]
&\Rightarrow& S^2 = 2 + \sqrt{2 + \sqrt{2 + \sqrt{2 + \sqrt{2 + \cdots}}}} 
				&\qquad \qquad \mathrel{\#} \text{square both sides} \\
[2pt]
&\Rightarrow& S^2 = 2 + S 
				&\qquad \qquad \mathrel{\#} S = \sqrt{2 + \sqrt{2 + \sqrt{2 + \sqrt{2 + \cdots}}}}  \\
[7pt]
&\Rightarrow& S^2 -S - 2 = 0 
				&\qquad \qquad \mathrel{\#} \text{collect terms} \\
[5pt]
&\Rightarrow& (S - 2)(S + 1) = 0
				&\qquad \qquad \mathrel{\#} \text{factor} \\
[5pt]
&\Rightarrow& S \in \{2,-1\}
				&\qquad \qquad \mathrel{\#} \text{solve for $S$} \\
[5pt]
&\Rightarrow& S = 2
				&\qquad \qquad \mathrel{\#} \text{positive root of $S$ (what about the negative root?)} 

\end{array}
\end{equation*}

\smallskip
\noindent
So apparently $ \sqrt{2 + \sqrt{2 + \sqrt{2 + \sqrt{2 + \cdots}}}}  = 2$.


\section{Does This Series Really Converge?}
It is worth asking whether this series really converges (and to what)
since using an algebraic variable, in this case $S$, to represent an 
infinite sum can go badly wrong \cite{can_you_spot_the_error}. 
This is because an infinite sum is really the limit of partial
sums: 

\begin{equation*}
\sum\limits_{i=1}^{\infty }a_{i} = \lim _{n\to \infty }\sum _{i=1}^{n}a_{i}
\end{equation*}

\bigskip
\noindent
and so can behave differently than a finite sum. 


\noindent
Maybe the first thing to notice here is that the $n^{\text{th}}$
term in this series, $a_n$, is

\bigskip
\begin{equation}
a_n = \underbrace{\sqrt{\sqrt{\sqrt{\cdots \sqrt{2}}}}}_\text{$n$ times} = 2^{{ \left (\tfrac{1}{2} \right )}^{n}}
\label{eqn:a_n}
\end{equation}

\bigskip
\noindent
Using Equation (\ref{eqn:a_n}) we can rewrite $\sqrt{2 + \sqrt{2
+ \sqrt{2 + \sqrt{2 + \cdots}}}}$ as the sum 

\bigskip
\begin{equation}
\sum\limits_{n=1}^{\infty} a_n = \sum\limits_{n=1}^{\infty} 2^{  {\left ( \tfrac{1}{2} \right )}^{n}}
\label{eqn:a_n_limit}
\end{equation}

\medskip
\subsection{Applying the Ratio Test}
\setstretch{1.35}
\noindent
So we are interested in whether sum in Equation
(\ref{eqn:a_n_limit}) converges and if so, to what.  One of the
easiest ways to test for convergence is the ratio test
\cite{wiki:ratio_test}, which says that if
$\lim\limits_{n \to \infty} \tfrac{a_{n+1}}{a_n} < 1$ the series
converges, if $\lim\limits_{n \to \infty} \tfrac{a_{n+1}}{a_n} > 1$ 
the series diverges and if $\lim\limits_{n \to \infty}
\tfrac{a_{n+1}}{a_n} = 1$ the test inconclusive.

\setstretch{1.0}

\bigskip
\noindent
In this case


\begin{equation*}
\begin{array}{llll}
\dfrac{a_{n+1}}{a_n}
&=&  \dfrac{2^{{ \left ( \tfrac{1}{2} \right ) }^{n+1}}} {2^{{ \left (\tfrac{1}{2} \right )}^n}} 
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} a_n = 2^{{\left (\tfrac{1}{2} \right )}^n} \\
[20pt]
&=& 2^{\left [{ \left (\tfrac{1}{2} \right )}^{n+1} - \, {\left (\tfrac{1}{2} \right )}^n \right ] }
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} \text{simplify: $\dfrac{x^a}{x^b} = x^{a-b}$} \\
[10pt]
&=& 2^{\left [ { \left (\tfrac{1}{2} \right )}^{n} \cdot \,  {\left (\tfrac{1}{2} \right )}^{1}  -\, {{ \left ( \tfrac{1}{2} \right )}^n}\right ] }
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} {\left (\tfrac{1}{2} \right )}^{n+1} = {\left (\tfrac{1}{2}\right )}^{n} \cdot {\left (\tfrac{1}{2}\right )}^{1}  \\
[10pt]
&=& 2^{\left [ {\left (\tfrac{1}{2} \right )}^{n} \cdot \,  \left ( \tfrac{1}{2} -1 \right ) \right ]} 
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} \text{factor out ${\left (\tfrac{1}{2} \right )}^n$} \\
[10pt]
&=& 2^{\left [ { \left (\tfrac{1}{2} \right )}^{n} \cdot \, \left ( - \tfrac{1}{2} \right ) \right ]} 
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} \tfrac{1}{2} - 1 = - \tfrac{1}{2}  \\
[10pt]
&=& 2^{- \left [ { \left (\tfrac{1}{2} \right )}^{n} \cdot  \, \left (\tfrac{1}{2} \right ) \right ]} 
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} \text{simpify}  \\
[10pt]
&=& 2^{- {\left (\tfrac{1}{2} \right )}^{n+1}}
				&\quad \qquad \qquad \qquad \qquad \mathrel{\#} x^a \cdot x^b = x^{a+b}
\end{array}
\end{equation*}


\bigskip
\noindent
So in the limit we have 


\begin{equation*}
\begin{array}{llll}
\lim\limits_{n \to \infty} \left [  \dfrac{a_{n+1}}{a_n} \right ]
&=&
\lim\limits_{n \to \infty} \left [  \dfrac{2^{{ \left ( \tfrac{1}{2} \right ) }^{(n+1)}}} {2^{{ \left (\tfrac{1}{2} \right )}^n}}  \right ]
				&\qquad \qquad \quad \mathrel{\#} a_n = 2^{{ \left (\tfrac{1}{2} \right )}^{n}} \\
[12pt]
&=&  \lim\limits_{n \to \infty} 2^{- {\left (\tfrac{1}{2} \right )}^{n+1}} 
				&\qquad \qquad \quad \mathrel{\#} \text{see above: 
				$\dfrac{2^{{ \left ( \tfrac{1}{2} \right ) }^{n+1}}} {2^{{ \left (\tfrac{1}{2} \right )}^n}} 
				= 2^{- {\left (\tfrac{1}{2} \right )}^{n+1}}$} \\
[15pt]
&=&  \lim\limits_{n \to \infty}  2^{- {\left ( \tfrac{1}{2^{n+1}} \right )}} 
				&\qquad \qquad \quad \mathrel{\#} \text{simplify} \\
[18pt]
&=&  \lim\limits_{n \to \infty}  \dfrac{1}{2^{\left ( \tfrac{1}{2^{n+1}} \right ) }} 
				&\qquad \qquad \quad \mathrel{\#} x^{-a} = \tfrac{1}{x^a}  \\
[18pt]
&=&  \lim\limits_{n \to \infty}  \dfrac{1}{2^0}
				&\qquad \qquad \quad \mathrel{\#} \lim\limits_{n \to \infty} \dfrac{1}{2^{n+1}} = 0 \\
[18pt]
&=&  \lim\limits_{n \to \infty} 1 
				&\qquad \qquad \quad \mathrel{\#} 2^0 = 1 \text{ and }  \dfrac{1}{1} = 1  \\
[15pt]
&=& 1
				&\qquad \qquad \quad \mathrel{\#} \lim\limits_{n \to \infty} c = c

\end{array}
\end{equation*}

\bigskip
\noindent
Since $\lim\limits_{n \to \infty} 2^{- {\left (\tfrac{1}{2}
\right )}^{n+1}} \!\! = 1$ the ratio test is unfortunately
inconclusive.
