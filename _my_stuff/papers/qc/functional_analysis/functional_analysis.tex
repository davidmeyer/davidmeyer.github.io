\section{Introduction}
Functional Analysis is an interesting topic because, among other reasons, it sits between
infinite dimensional linear algebra and real and complex analysis. As we shall see later,
the connection is the norm $\left\lVert x\right\rVert$ of a vector space $V$. As we shall 
see, if $(V,\left\lVert \cdot \right\rVert)$ is a \emph{normed space} then $(V,d)$ is a
\emph{metric space} with metric $d(x,y) \coloneqq \left\lVert x-y \right\rVert$. Moreover, if 
$(V,\langle\cdot,\cdot\rangle)$ is an \emph{inner product space} (that is, 
$\langle\cdot,\cdot\rangle$ is an inner product on vector space $V$), then 
$(V,\left\lVert \cdot \right\rVert)$ is a normed space with norm
$\left\lVert x\right\rVert \coloneqq \sqrt{\langle x,x\rangle}$ and so $(V,d)$ is a
metric space with metric $d(x,y) = \sqrt{\langle x-y,x-y\rangle}$.  
Functional Analysis is inhabited by all kinds of interesting objects like these.

\bigskip
\noindent
To measure this distance we define a map 
$d: X \times X \rightarrow [0,\infty)$, which we 
call a metric and which has the 
following three properties:


\begin{equation*}
\begin{array}{llllll}
(a) & d(x,y) = 0 \Leftrightarrow x = y				&& \qquad \qquad \mathrel{\#} \text{this property is called "positive definite"} \\
[8pt]
(b) & d(x,y) = d(y,x)								&& \qquad \qquad \mathrel{\#} \text{this property is called "symmetric"} \\
[8pt]
(c) & d(x,y) \leq d(x,z) + d(z,y)					&& \qquad \qquad \mathrel{\#} \text{this is called "triangle inequality"}
\end{array}
\end{equation*}

\bigskip
\noindent
We can see that we want the distance between $x$ and itself to be zero, hence property $(a)$. However,
more than that we want this only to be the case when $x = y$ (so if and only if). Of course we want 
the distance from $x$ to $y$ to be the same as the distance from $y$ to $x$; this is property $(b)$. 
Finally,  we will see the triangle inequality (property $(c)$) in many forms but here we see that if 
you take a "detour", say through $z$, the detour path will be longer (have a greater or possibly equal 
distance) than the direct path between $x$ and $y$.
